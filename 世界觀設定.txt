================================
　　概念
================================

　銀器和甜點
　銀餐具能試毒
　銀餐具渴望甜點
　類似吸血鬼的感染/眷屬設定

　有點類似哨嚮或ABO那樣的世界觀
靈感來自叉糕的字面涵義，但跟叉糕內容完全無關
沒有R18G，但有些微R15內容

================================
　　甜點體質
================================

# 香氣的效果 #
這種體質的人身體會散發甜點香味，香味有分成三種
- 糖果香味：使人精神亢奮、充滿能量、爆發力增強
- 茶點香味：情緒緩和的效果、放鬆和愉悅、痛覺減緩
- 蛋糕香味：實際的提高了身體癒合能力，外傷更快的復原、免疫力提升
其中蛋糕香味的人最少(糖果14:茶點5:蛋糕1)

氣味對自己本身也是有效果的，最強烈的蛋糕香味的人很可能接近不死，只會自然老化死亡
一般人雖然能受到香氣的效果影響，被治癒或振奮，但他們並不會聞到甜點氣味
銀器體質的人也能受到香氣的效果影響，且能聞到他們的甜味

# 階級 #
甜點體質對人影響的效果並沒有本能的階級之分
但在社會對這些效果的需求影響下，蛋糕香味的人通常會更被人們需要，因此有些人會認為蛋糕香味的人階級更高
茶點香味次於蛋糕香味，糖果香味排在最後

# 經常能看到甜點體質的職位 #
- 糖果香味：能利用氣味振奮自己的隊友或下屬，多擔任領導者、警職、活動領隊
- 茶點香味：能緩和情緒、放鬆，多擔任諮商師、社工等照護與心理疏導的工作
- 蛋糕香味：治癒的氣味讓他們更好的擔任醫生、急救人員等工作

# 體質能力控制 #
當他們口中含有含糖的食物，平靜呼吸能夠自主散發出香氣
當他們感到安全，或者正向的情緒時，也會不由自主的散發出香氣

# 體質來源 #
甜點體質是天生的，但不是遺傳來的
很可能在家族中完全沒有甜點體質，卻突然在某一代的孩子身上有甜點體質出現

# 體質發現 #
通常只會由特殊體質的人發現某個人是甜點體質，一般人因為聞不出特殊甜味而很難發現甜點體質的人
特別強烈的蛋糕香味除外，因為治癒能力強烈到肉眼可見輕微外傷快速癒合，這種類型的甜點體質一般人也有可能發現

# 黑血中毒 #
甜點體質的人接觸到銀器體質的人的黑血會發燒或身體不適，在這期間會是虛弱的狀態
黑血中毒時身體會被黑血的所屬人吸引，即使精神上可能是不願意的，過一段時間就會從這個狀態自我痊癒，通常是兩個小時到三天之間

# 偽甜點 #
甜點體質的人為了避免自己被銀器體質的人傷害，他們的氣味有一種保護機制
能夠讓非特殊體質的人攝入自己的體液，使攝入者暫時成為偽甜點

================================
　　銀器體質
================================

# 感知力 #
這種體質的人感知力很強，他們可以自主的觸發強烈感知
也可能在精神狀態差或者精神劇烈起伏時，無法自控的觸發強烈感知，這是身體在感到危險時自動觸發的防衛機制

# 爆發感知 #
當他們的感知能力爆發時，感官會變得強烈
- 視覺：他們有夜視能力，眼睛在黑暗中會泛銀光，極遠的物體也能清楚辨識
- 聽覺：隔著牆也能清楚的聽見聲音，在水中也能辨識聲音的方向
- 嗅覺：能夠聞到酸、腐敗、致病、毒物等有害物的特殊味道，即使目標物本身是無味的
- 觸覺：他們能感知絕對溫度，可以說得出溫度確切的數值，能感覺得出空氣流動、水流的確切方向
- 味覺：對酸、辣、涼等刺激性的味道會放大感知，吃下這些味道的食物會強烈到暫時麻痺了各種感官

一旦他們進入爆發感知狀態，任何使他們不適的感知都會觸發身體的試毒反應

過度使用可能會讓身體非常疲倦，有可能在結束後倒地不起
如果都沒有遇到不適的感知，結束時他們精神會是正常的，可能還會很有精神，雖然身體可能會很疲倦
甚至會有舒暢和愉悅感，就像剛運動完那樣

# 謊言具象化 #
當說話者的思考與說出的話有異時，謊言會具象成感知
例如看見黑色霧狀從說謊的人口中流出飄散到空氣中
或者說話者的聲音夾雜刺耳電器雜音
具象化的方式通常是令人不舒服的(例如臭味或皮膚刺癢)

同一人通常只會有一種具象化方式，如果他是看見謊言，就不會是聽、聞到謊言
自己說出的謊言也會具象化
這個感知是常態的，即使不在爆發感知的狀態也能感受到
但只有在爆發感知的狀態下會引起試毒反應

銀器體質的人通常會對此謊言感到排斥
因為謊言具象化的方式通常讓他們感到身體上的不適，且可能會觸發試毒反應
但也有不排斥謊言的例外

當他們在爆發感知時，接觸具象化的謊言，會觸發試毒反應

# 試毒反應 #
爆發感知期間接觸到不適感知時、感知到謊言時，這兩種情況都會觸發這個體質的人的試毒反應
他們的眼睛會隨著感知量慢慢轉變成黑色，就好像一切都是有毒的，讓銀器漸漸失去光澤
眼睛轉變成黑色的過程中，身體會慢慢失去活力，精神狀態會變差
有可能因精神狀態差，造成不自主的爆發感知，陷入惡性循環

# 中毒狀態 #
當眼睛完全轉變成黑色，他們的血也會變成黑的，這是銀器體質的人的中毒狀態
體力會反彈般恢復活力，即使身體已經無法負荷也能靈活行動
且能暫時讓體能突破平常的上限，例如誇張到非人類的耐力與力量
判斷能力和自制力下降，對甜點體質的人的渴望變強烈

# 渴望 #
他們對甜點體質的人的體液有本能的渴望，尤其在他們越接近中毒狀態時渴望越強
如果他們能食用甜點體質的體液，可以減輕試毒反應，恢復體力，讓眼睛變回常態的顏色，避免進入中毒狀態

即使他們沒有觸發任何試毒反應，普通的疲憊狀態下也會對甜點體質的人有想要吃掉的慾望
壓抑這個慾望的話會滾雪球般越來越強烈

# 體質來源 #
銀器體質可能是天生或後天感染的，且起初都會是未活化的狀態
銀器體質不論有沒有活化，都很可能遺傳到後代，也可能在完全沒有銀器體質的家族中出現

# 體質活化 #
未活化時他們就像普通人一樣，也有些人終生都沒有被發現自己是銀器體質，本人也可能不自知
即使未活化，一般食物的甜味對他們也有吸引性，也有不被一般食物甜味吸引的例外
只有在他們第一次嘗到甜點體質的人的體液時，他們的銀器體質才會活化

# 啟蒙者 #
初次嘗到某個甜點體質的人的體液，這個人將會是他的啟蒙者
銀器體質的人無論在怎樣的狀態下，都會對自己的啟蒙者有強烈的進食慾望

# 黑血 #
中毒狀態時血液變成黑色，這個黑血會讓銀器體質變成有傳染性
普通人攝入黑血會被轉化為銀器體質，剛轉化時會是未活化狀態
甜點體質的人攝入黑血會虛弱發燒，且被黑血來源的銀器體質人吸引，持續時間視感知力影響，感知力越強持續越久，通常在兩小時到三天之間
這是銀器體質的人保命的機制，出現黑血時銀器體質的人會非常需要甜點體質的人的體液，借由這個吸引力讓甜點體質的人為他提供體液

# 與其他同體質人 #
銀器體質的人互相之間會有自然而然的階級感，他們能直覺的認知互相的感知力強度
感知力較弱的那方會本能的順從或跟隨感知力較強的那方
這個順從與跟隨不受是否活化的影響，因此有可能發生已活化的人下意識的尊敬未活化的人，且認知以為是單純仰慕這個人的其他能力(例如工作效率、更好的體能)
也存在少數排斥同體質人的例外

================================
　　非特殊體質 - 一般人
================================

# 關於香氣 #
通常情況下聞不到甜點體質的人的香氣，但能受惠於氣味的效果

# 黑血感染 #
如果攝入銀器體質人的黑血，會被轉化為銀器體質
但起初會是未活化狀態，很可能因此沒有發現轉化

# 偽甜點 #
一般人如果攝入甜點體質的人的體液，會使攝入者暫時成為偽甜點
偽甜點將會散發與來源人相同的氣味
偽甜點的狀態下，能夠聞到所有甜點體質的人散發的香氣

偽甜點仍然會被黑血感染，這會讓偽甜點在轉化當下就被活化，這時香氣來源人會成為他的啟蒙者
來源人的甜點香味越強烈，偽甜點的持續時間越長，最短在兩小時左右，最長能夠持續三天

偽甜點可能會被誤認為甜點體質的人，但他們只有香氣，而沒有治癒的效果
偽甜點可能會被銀器體質的人當作進食對象

================================
　　衍伸設定
================================

# 規定 #
銀器的進食是食用甜點的體液，這仍然可能造成某些體液傳染的疾病散播，因此有些抑制藥物和甜點體質替代物被開發出來
銀器體質在某些地區被視為無法完全治療的疾病，被規定要長期服用抑制甜食慾望的藥物，也可能連一般食物的甜食都被禁止食用
避免疾病因此散播，在現代也有禁止私下交易甜點體質體液的相關法律，只能透過相關機構的捐贈系統來檢查並配給已經登記的銀器體質人使用
一些地區可能更極端的禁止銀器體質的人進入爆發感知狀態，看到眼睛出現銀光就懲罰

# 傳說 #
一些古早傳說中的神明很可能是真實存在的人物，他們很可能是特殊體質的人，例如治百病的神仙其實是香氣效果特別強烈的甜點體質人
造成村莊浩劫的惡魔的傳說也可能是黑血狀態的銀器體質人
靠幾乎非人類的身體素質保護了人民的古代傳說也可能是銀器體質人
